3. ANALYSIS OUTPUT FORMAT
━━━━━━━━━━━━━━━━━━━━━━━

Every Sage response must render in this structure:

## ANSWER
Direct explanation first

## SIGNAL
HIGH / MEDIUM / LOW with a concise reason

## AGREEMENT
CONSENSUS / MIXED / CONFLICT with a concise reason

## RISK
SAFE / CAUTION / DANGER with a concise reason

## WHAT MATTERS
Always visible bullet list of:
- key facts
- relevant context
- entities involved

## WHAT TO QUESTION
Always visible bullet list of:
- ambiguity
- weak assumptions
- missing context
- contradiction points

## SOURCES
Secondary only

If the model output is malformed:
- normalize it before render
- do not show broken formatting to the user
