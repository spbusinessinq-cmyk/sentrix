You are a senior product designer and front-end engineer building a production-grade browser concept called Vero.

IMPORTANT:
This is NOT a generic SaaS dashboard.
This is NOT a hacker toy UI.
This is NOT a military simulator.
This is a consumer-friendly tactical browser interface with a sharp, glassy, Splinter Cell-inspired design language.

CORE PRODUCT:
- Product name: Vero
- Security engine: BLACKDOG
- Positioning: a safer, cleaner browser with visible threat awareness
- BLACKDOG is the embedded security layer
- AXION is NOT connected
- This is a public-facing browser product for everyday users

DESIGN DIRECTION:
Build a dark, premium, sharp, glassy interface that feels:
- tactical
- user friendly
- controlled
- modern
- layered
- fast
- serious

Aesthetic reference:
“Splinter Cell tactical UI made clean enough for public consumer use”

VISUAL RULES:
- Very dark background palette
- Thin borders
- Soft glass panels
- Subtle glow, never cheesy neon overload
- Sharp spacing and compact structure
- Dense but readable
- Professional typography
- Smooth hover states and transitions
- No cartoonish UI
- No oversized rounded bubbly components
- No generic startup gradients

COLOR LOGIC:
- Base: near-black / charcoal
- Panels: dark glass
- Accent green = safe / protected / connected
- Amber = caution / moderate risk
- Red = danger / blocked / high risk
- Blue = neutral system info
Keep accents controlled and meaningful.

BUILD GOAL:
Create a polished browser interface prototype in React that looks like a real downloadable browser product.

TECH STACK:
- React
- TypeScript
- Tailwind CSS
- Single-page app
- Production-quality component structure
- Responsive enough for desktop presentation
- Clean code organization

MAIN LAYOUT:
Build a full browser shell with these regions:

1. TOP TAB BAR
- Vero wordmark/logo area
- Multiple tabs
- Active tab styling
- New tab button
- BLACKDOG status indicator on the right
- Clean, premium browser feel

2. ADDRESS / SEARCH BAR ROW
- Back / forward / refresh controls
- Main address bar
- Current domain shown

  - SAFE
  - CAUTION
  - DANGER
  - UNKNOWN

3. LEFT TACTICAL SIDEBAR

- Home
- Search
- History

- Vault
- Settings

- Hero search field
- “Protected by BLACKDOG” indicator
- Quick actions cards:
  - New Secure Tab
  - Burn Session
  - Link Check
  - Privacy Report

5. RIGHT BLACKDOG PANEL

A slide-out / fixed tactical analysis panel showing:
- BLACKDOG ACTIVE
- Trackers blocked
- Scripts flagged
- Redirects stopped
- Page risk score
- Session status
- Small live event log

- result title
- domain
- summary snippet
- risk label
- short BLACKDOG explanation

- SAFE: low trackers, verified domain
- CAUTION: mixed reputation, third-party scripts
- DANGER: suspicious redirects, deceptive download pattern
